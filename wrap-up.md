## Questions

### Please provide instructions on how to run your project in a bulleted list below.
- in root folder run "docker-compose up -d" to update the database

- "cd ./client" to enter the client folder 
- run "npm install"
- in client folder run "npm run start" to run the React app

- open a new termnial window at the root
- "cd ./server" to enter the server folder
- run "npm install"
- in server folder run "npm run start" to runthe Node/Express app

- open localhost:3000 in the browser
